# 02. ICP · 독자

## 핵심 독자 1줄
스포츠 경기·선수·기록을 좋아하지만, 검색하면 숫자·규칙이 출처마다 달라 헷갈려서 "정확히 정리된 한 편"을 찾는 한국 팬.

## 들어오는 검색 의도
- 규칙/구조 이해 — "오프사이드 예외", "포메이션 상성", "분배금 어떻게 정해지나".
- 숫자 확인 — "우승 상금 얼마", "중계권 분배", "연봉/상금 비교".
- 비교 — "분데스리가 vs EPL", "리그별 상금".
- 입문 가이드 — "골프 첫 라운드", "마라톤 입문".

## 독자 페인포인트 5개
1. 검색하면 글마다 숫자가 달라서 뭐가 맞는지 모르겠다.
2. 영어·전문 용어를 그대로 옮겨놔서 안 읽힌다.
3. 단편적 사실만 있고 "그래서 구조가 어떻다"는 큰 그림이 없다.
4. 출처 없이 단정해서 믿어도 되는지 불안하다.
5. 글이 길기만 하고 정작 궁금한 답이 위에 없다.

## 독자가 결정 직전 궁금해하는 것
- "이 숫자, 어디까지 포함한 거야?" (범위 구분)
- "그래서 결론이 뭔데?" (3줄 요약 욕구)
- "이거 진짜 맞아?" (출처·추정 표기)

## 읽고 나서 바라는 행동
- 제휴 미설정 상태 → 지금은 **버튼·구매 유도 금지**.
- 목표는 "정확히 알았다"는 만족 → 내부링크로 다른 글 회유 → 재방문.
