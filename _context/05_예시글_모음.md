# 05. 예시 글 모음 (가장 강력한 레퍼런스)

킹덤 톤이 잘 적용된 글을 여기에 모은다. 새 글 쓸 때 "이 톤·문장 길이·구조를 almost mimic 하라"고 지시.

## 기준 예시 (페르소나 적용 1호)
- **`src/content/blog/bundesliga-prize-money.md`** — '킹덤' 페르소나 + 1인칭 변수를 처음 입힌 글.
  - 본받을 점: 도입을 "검색하면 숫자가 다 달라 헤맸다"는 1인칭으로 열고, 중간에 "직접 표로 묶어봤다 / 중계로 봤다" 경험을 섞음. 사실·수치는 전부 추정 표기 유지.
  - 새 글은 이 글의 문장 길이·박스/표/FAQ 리듬을 그대로 따른다.

## 추가 예정
- 페르소나로 새로 쓰거나 리라이트한 글이 나오면 여기에 경로를 계속 추가한다.
- 가장 잘 나온 2~3편만 유지(너무 많으면 톤이 흐려짐).
